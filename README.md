
  # Enterprise Landing Page Design for Appointy

  A single-page enterprise landing experience built with React, Vite, and Tailwind CSS v4. The layout is composed of modular sections that showcase enterprise scheduling value props, use cases, integrations, and CTAs.

  ## Page Sections

  - Navigation (`src/app/components/Navigation.tsx`)
  - Hero (`src/app/components/HeroSection.tsx`)
  - Horizontal feature carousel (`src/app/components/HorizontalScrollSection.tsx`)
  - Stacked feature cards (`src/app/components/StackedCardsSection.tsx`)
  - Use cases, consultation, integrations, final CTA, team (`src/app/components/ContentSections.tsx`)

  ## Tech Stack

  - React 18 + TypeScript
  - Vite 6
  - Tailwind CSS v4 (via `@tailwindcss/vite`)
  - UI utilities and libraries (e.g., Radix UI, lucide-react, class-variance-authority)

  ## Project Structure

  ```text
  src/
  ├─ app/
  │  ├─ App.tsx
  │  └─ components/
  ├─ assets/
  ├─ imports/
  ├─ main.tsx
  └─ styles/
     ├─ fonts.css
     ├─ index.css
     ├─ tailwind.css
     └─ theme.css
  ```

  ## Development

  - Install dependencies: `npm install`
  - Start dev server: `npm run dev`
  - Build for production: `npm run build`
